# 🏥 Patient Readmission Analysis

## 📌 Objective
To analyze hospital readmission patterns and identify high-risk patients.

## 🛠️ Tools Used
- SQL
- Excel
- Power BI

## 📊 Dataset
Contains patient demographics, diagnosis, readmission status, and length of stay.

## 🔍 Key Analysis
- Readmission rates
- Diagnosis-wise distribution
- Average hospital stay
- High-risk patient identification

## 📈 Insights
- Patients with chronic diseases have higher readmission rates
- Longer stays often indicate higher readmission risk
